#  Local Food Wastage Management System

An advanced data analytics and operational dashboard built to tackle the critical issue of food wastage. This platform connects surplus food providers (restaurants, supermarkets) with receivers (NGOs) while providing deep insights into donation trends.

##  Key Features
- **Analytics Hub:** Real-time KPIs tracking active listings, total volume, and claim success rates.
- **Advanced Visualizations:** Interactive Plotly charts (Pie, Bar) for demand and supply analysis.
- **Data Explorer:** A robust directory to search and filter active food listings and provider contacts.
- **Complete CRUD Interface:** Secure database management system to Create, Read, Update, and Delete food records dynamically.

##  Tech Stack
- **Frontend:** Streamlit
- **Backend/Database:** SQLite (SQLAlchemy)
- **Data Manipulation:** Pandas
- **Data Visualization:** Plotly Express

##  Business Impact
This project identifies key wastage patterns (e.g., highest wastage by meal types and cities) and streamlines the distribution supply chain to ensure zero-hunger goals are met efficiently.
